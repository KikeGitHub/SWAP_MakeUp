# Galería de Trabajos

Coloca aquí las imágenes de tus trabajos de maquillaje.

## Imágenes Incluidas en el Código:
- **work-1.jpg** - Maquillaje de novia
- **work-2.jpg** - Maquillaje social
- **work-3.jpg** - Maquillaje editorial
- **work-4.jpg** - Maquillaje artístico
- **work-5.jpg** - Maquillaje de noche
- **work-6.jpg** - Maquillaje natural

## Especificaciones:
- **Tamaño recomendado**: 800x800px (cuadradas)
- **Formato**: JPG
- **Calidad**: Alta resolución pero optimizada para web
- **Cantidad**: Mínimo 6, puedes agregar más

## Para Agregar Más Imágenes:

1. Agrega las imágenes a esta carpeta con nombres como `work-7.jpg`, `work-8.jpg`, etc.

2. Edita `index.html` y agrega más elementos en la sección de galería:

```html
<div class="gallery-item" data-aos="fade-up" data-aos-delay="100">
    <img data-src="gallery/work-7.jpg" alt="Descripción del trabajo" class="lazy-load">
    <div class="gallery-overlay">
        <i class="fas fa-search-plus"></i>
    </div>
</div>
```

## Consejos para las Fotos:
- Usa buena iluminación
- Muestra diferentes estilos de maquillaje
- Incluye antes/después si es posible
- Enfócate en el resultado del maquillaje
- Mantén un estilo visual consistente
