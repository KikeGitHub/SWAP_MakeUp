# Imágenes Requeridas

Coloca las siguientes imágenes en esta carpeta:

## Imagen Hero
- **hero-bg.jpg** - Imagen de fondo para la sección hero (1920x1080px recomendado)
  - Debe ser una imagen impactante de maquillaje o la artista trabajando
  - Alta resolución y buena iluminación

## Foto Sobre Mí
- **about-photo.jpg** - Foto profesional del artista (800x1000px recomendado)
  - Retrato profesional
  - Fondo elegante o estudio

## Redes Sociales
- **og-image.jpg** - Imagen para compartir en redes sociales (1200x630px)
  - Logo + foto del artista o trabajos destacados

## Favicon
- **favicon.png** - Icono del sitio (32x32px o 64x64px)
  - Logo o iniciales del negocio

## Productos (4 imágenes)
- **product-1.jpg** - Base de maquillaje (500x500px)
- **product-2.jpg** - Paleta de sombras (500x500px)
- **product-3.jpg** - Labiales (500x500px)
- **product-4.jpg** - Set de brochas (500x500px)

## Notas:
- Usa imágenes de alta calidad
- Optimiza para web (comprime sin perder calidad)
- Formatos recomendados: JPG para fotos, PNG para elementos con transparencia
- Puedes usar herramientas como TinyPNG para comprimir
